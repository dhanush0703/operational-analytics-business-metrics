# Operational Analytics: Investigating and Improving Business Metrics

## Project Overview
This project focuses on analyzing operational and user engagement metrics using SQL-based analysis to identify inefficiencies, trends, and opportunities for business improvement.

## Key Objectives
- Analyze operational throughput and job review activity
- Investigate user engagement, retention, and growth trends
- Identify metric spikes, data quality issues, and behavioral patterns
- Derive actionable insights for business decision-making

## Key Analyses Performed
- Jobs reviewed over time and throughput analysis
- 7-day rolling average performance metrics
- Language share distribution analysis
- Duplicate row detection and data quality checks
- Weekly user engagement and retention analysis
- User growth and email engagement analysis

## Tech Stack
- SQL
- MySQL Workbench
- Data Analysis & EDA
- Canva (Reporting & Visualization)
